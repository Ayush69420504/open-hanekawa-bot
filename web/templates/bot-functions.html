Functions

Synatx: !ara [function] [function args ..]

  1. General Functions
    • whoami

      • = Returns an embed containing bot information

    • ping

      • = Returns the current bot latency in ms

    • hostinfo

      • = Returns important system information from host machine

  2. QRCODE Generation using google apis
    • urltoqr

      • = Takes a url as input and provides a googleapi link of a qrcode leading to that url

    • wifitoqr

      • = Takes ssid, encryption and password as input and produces a qrcode with which android phones can connect to that network

  3. Manga functions
    • search-manga

      • = Requires one argument: Name of manga. Returns paginated embed(s) containing information about the manga'(s)

    • list-chapters

      • = Requires manga-code, starting chapter, ending-chapter. If no starting and ending chapter is provided then the entire chapter list is provided . If only starting chapter is provided then it shows all chapter from that chapter. If both starting and ending chapter are provided then it shows the list in that range

    • download-chapter

      • = Requires manga-code and chapter-code. Returns a pdf of that chapter

  4. Music functions
    Providers: Soundcloud and Lofi Girl
    • search-song

      • = Requires keyword(s). Returns 5 paginated embeds relevant to the song

    • joinvc

      • = Joins the current voice channel of the user. Throws a error message if user is not in a voice channel

    • leavevc

      • = Leaves the current voice channel of that guild

    • pause

      • = Pauses the ongoing audio

    • resume

      • = Resumes the paused audio

    • now-playing

      • = Returns an embed containing the name, current-time, duration and thumbnail of the current audio

    • skip-track

      • = Skips the current playing or paused track

    • list-queue

      • = Shows the songs collected in the queue for that particular guild

    • lofibeats

      • = Starts playing a hls stream of Lofi Girl from chilled cow

    • stoplofi

      • = Stops the lofi hls player and allows queue system to work

    • queue

      • = Takes a keyword(s) as an argument. Adds a song to the queue, and starts playing it, if no song is playing

    • drop-queue

      • = Drops the entire queue for that particular guild/channel. Basically a hard reset
